将设置界面重构为现代化的左右分栏布局，以支持更丰富的配置选项和更好的扩展性。

## 1. 样式更新 (App.css)
添加新的 CSS 类以支持分栏布局：
- `.settings-layout`: Flex 容器，包含侧边栏和内容区。
- `.settings-sidebar`: 左侧导航栏，固定宽度，包含导航项。
- `.settings-content`: 右侧内容区，自适应宽度，显示具体配置。
- `.settings-nav-item`: 侧边栏导航项样式（激活/悬停状态）。
- 调整模态框尺寸为更大、更宽的尺寸。

## 2. 重构 SettingsModal 组件
- **布局变更**：将 Modal 内容区域改为 `.settings-layout`。
- **左侧导航**：
  - 添加 "Agents" 选项（作为默认选中项）。
  - 预留 "General" 等其他选项的扩展空间。
- **右侧内容**：
  - **Agent 选择器**：将原来的 Tab/Button 切换改为 `<select>` 下拉列表，位于内容区顶部。
  - **动态渲染**：根据下拉列表选中的 Agent，渲染对应的配置表单（安装状态、命令、环境变量等）。
  - 复用并适配现有的配置逻辑。

## 3. 交互逻辑
- 用户点击设置 -> 打开大尺寸模态框。
- 默认选中左侧 "Agents"。
- 右侧默认显示当前使用的 Agent（或第一个），可通过下拉框切换。
- 切换 Agent 后，界面实时更新为该 Agent 的配置信息。
